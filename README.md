# Compiler Pipeline

## 1. Lexical Analysis

The lexer converts source code into tokens.

Example:

```text
let x=10;
```

becomes:

```text
LET IDENTIFIER EQUAL NUMBER SEMICOLON
```

---

## 2. Parsing

The recursive descent parser converts tokens into an Abstract Syntax Tree (AST).

---

## 3. Bytecode Generation

The compiler traverses the AST and emits bytecode instructions.

Example:

```text
CONSTANT
ADD
PRINT
```

---

## 4. Virtual Machine Execution

The stack-based VM executes bytecode instructions using:

- Instruction Pointer (IP)
- Operand Stack
- Global Variable Table
- Dispatch Loop

The disassembler converts bytecode into a human-readable representation.

# VM Trace Mode

Trace mode visualizes VM execution step-by-step.

## Example

```bash

```

### Sample Trace

